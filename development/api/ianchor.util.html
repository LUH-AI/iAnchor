ianchor.util.pytorch_image_wrapper(device=None)[source]¶

Decorator that converts anchor image samples (np.ndarray) to a Tensor and extracts the labels (with argmax). Permuted the images to AxHxW before passing it to the wrapped method.

Use case:

Adapter from Anchor Lib -> predict_fn -> Anchor Lib
